AUTOUNIT 을 소개합니다

AutoUnit은 반응형 웹 페이지 개발자 및 디자이너를 위한 보조 솔루션 입니다, 이제 더이상 귀찮은 계산과정들을 직접하지 말고 AutoUnit에게 맡겨보세요.

계산 가능한 단위들

AutoUnit은 아래의 값들을 계산하기에 적합합니다 사용하기전에 자신이 원하는 단위가 있는지 한번 확인 해보세요.

  1. PX
  2. VW
  3. VH
  4. REM
  5. EM
  6. %

사용하는 방법

자 이제 시작을 위해 사용하는 방법을 알아보겠습니다, 만약 당신이 프로토타입으로 포토샵과 같은 프로그램에서 디자인한 최대 너비 1920px의 웹 페이지에서 100px의 모양이 얼마만큼의 vw을 가지는지 계산하는 과정은 아래와 같습니다.

  1. 원하는 단위를 vw로 선택합니다.
  2. 베이스 값에 1920(px)을 입력 합니다.
  3. 변환하고 싶은 값을 100(px)로 입력 합니다
  4. VW/H 변환에 체크가 되어있는지 확인 합니다.
  5. 결과창에 있는 계산하기를 눌러 값을 확인합니다.

다음으로는 16px을 1rem으로 기준을 잡은 페이지에서 500px이 얼마만큼의 rem을 가지는지 계산하는 과정 입니다.

  1. 원하는 단위를 rem로 선택합니다.
  2. 부모 혹은 루트 값에 16(px)을 입력 합니다.
  3. 변환하고 싶은 값을 500(px)로 입력 합니다
  4. 결과창에 있는 계산하기를 눌러 값을 확인합니다.

마지막으로 단축키를 사용하여 16px을 1rem으로 기준으로 잡은 상황에서 31.25rem의 실제 px값을 알아보는 과정 입니다.

  1. 단축키Q 를 눌러서 원하는 단위를 선택합니다.
  2. 변환값이 R/EM이 아니라면 Q를 한번더 누릅니다.
  3. R을 눌러 부모 혹은 루트값에 16(px)을 적습니다.
  4. T를 눌러서 31.25(rem)을 적고 엔터키를 누릅니다.

단축키 모음

혹시 더욱 빠른 작업을 원하시나요? 그러한 당신을 위한 도움책으로 단축키를 드리겠습니다, 다음의 단축키들은 여러분의 작업속도의 효율을 더욱 증가시켜줄 것 입니다. 각각의 키들을 눌러보면서 숙달해보세요.

단축키 설명

W 원하는 단위를 VW단위로 변경합니다.
A 원하는 단위를 VH단위로 변경합니다.
S 원하는 단위를 REM단위로 변경합니다.
Z 원하는 단위를 EM단위로 변경합니다.
X 원하는 단위를 %단위로 변경합니다.
B 입력칸에 베이스 값을 선택합니다.
R 입력칸에 부모 혹은 루트값을 선택합니다.
T 입력칸에 변환하고 싶은 값을 선택합니다.
